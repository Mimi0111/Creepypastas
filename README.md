# Creepypasta — GitHub Pages static site

Sitio estático (single-page) para publicar y leer creepypastas.
Características:
- Pantalla de inicio retro con "Creepypasta" y "Created by: Mimi".
- Leer historias guardadas localmente (LocalStorage).
- Agregar historias con texto, foto, video y audio (previsualización).
- Opción opcional para publicar directamente a un repositorio de GitHub mediante la API (requiere Personal Access Token con permisos para crear archivos).

Instrucciones:
1. Abre `index.html` en tu navegador (o sube todo a un repo y activa GitHub Pages).
2. Para publicar en GitHub:
   - Crea un Personal Access Token con `repo` scope.
   - Rellena Owner, Repo, Path y Token en el formulario antes de publicar.
   - Atención: no compartas tu token públicamente.

Este paquete está diseñado para ser sencillo y fácil de adaptar.
